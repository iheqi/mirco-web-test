# micro_web_test

#### 启动主应用
```shell script
cd main # 进入到主项目
npm run start # 启动项目
```

#### 启动子应用
micro_web_test 目录下运行：

```shell script
npm start 
```
包括service应用也会启动。

#### 项目介绍

- main 主项目，包含框架代码，位于`micro`目录下。
- 其他为子项目代码。 


#### 子项目分类
> 我的。计算器，关注度排行，上市新车 react16

> 本地车市、首页、搜索、选车、经销商、新能源、车型   vue2

> 车系、对比、询价页、金融购车、降价频道  vue3

> 资讯、视频  react15
